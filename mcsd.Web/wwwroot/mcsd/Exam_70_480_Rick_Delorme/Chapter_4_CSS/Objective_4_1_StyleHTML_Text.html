Text Aligment

In CSS, you can align text in various ways using the text-align property. Here are some common options:

- Left-align text : 'text-align: left;'
- Center-align text : 'text-align: center;'
- Right-align text : 'text-align: right;'
- Justify text : 'text-align: justify;'

You can apply the text-align property to various HTML elements like p, h1, h2, div, span, etc.
